# LoftRES

Ok, maybe this is the most interesting tool for Ravenloft: Strahd's Possession and Ravenloft: Stone Prophet videogames. With this tool you can extract/generate resource (RES??) raw files AND you can decrypt/convert, that raw files into .TGA indexed files and viceversa. The new converted raw image will be inserted in the RES?? file WITHOUT compression.
